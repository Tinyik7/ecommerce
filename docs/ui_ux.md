## 1. Прототип

- Основной интерактивный прототип реализован прямо в Flutter (экраны `lib/app/modules/*`), что позволяет демонстрировать сценарии на симуляторе/устройстве.  
- Внешний визуальный стиль: Poppins, палитра `#1F41BB` (primary) + нейтральные оттенки, иконки в `assets/vectors`.  
- Для UI-итераций использован рабочий макет в Figma (см. ссылку в презентации), где совпадают цвета, spacing и состояния кнопок → проще обсуждать правки с дизайнером.

## 3. UX-сценарии

| Сценарий | Шаги | Особенности UX |
| --- | --- | --- |
| «Быстрый просмотр / покупка» | Запуск → Каталог → Поиск «куртка» → Сортировка по цене ↓ → Карточка → «В корзину» → Корзина → «Купить» | Поиск реагирует на ввод, фильтры кликабельные, покупка оформляется сразу даже офлайн. |
| «Вахтовый режим» | Интернет пропал → Каталог показывает кешированные карточки + баннер «оффлайн» → пользователь добавляет товар → корзина хранит локально до синка | Сообщения об офлайне, данные из SQLite, синк при восстановлении сети. |
| «Смена языка» | Настройки → Язык → `ru/en/kk`, мгновенный `Get.updateLocale` | Все строки вынесены в `translations.dart`, применяется без перезапуска. |
| «Поддержка» | Настройки → Помощь → диалог с контактами | Важное требование для предзащиты — наличие канала обратной связи. |

## 4. Навигация и логика переходов

- Центральная точка — `BaseView` с `PageView`/BottomNavigation (GetX).  
- Роутинг описан в `lib/app/routes/app_pages.dart`, есть Splash → Auth → Base.  
- Для детальных экранов (`ProductDetailsView`) используется именованный маршрут + аргументы.  
- Навигация между состояниями (например, после `logout`) проходит через `Get.offAllNamed`, чтобы исключить возврат на защищённые экраны.

## 5. UI-структура и стиль

- Компонентный подход: карточки (`components/product_item.dart`), кнопки, элементы настроек.  
- Единая система отступов (`flutter_screenutil`) → адаптация под разные диагонали.  
- Анимации (`flutter_animate`) добавлены на карточки и кнопки для плавности.  
- Цвета/шрифты централизованы в `config/theme/my_theme.dart`.
